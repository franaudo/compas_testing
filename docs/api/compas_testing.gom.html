compas_testing.gom

Import/Export

results_to_list

Converts the text results to a list and computes the geometric keys of the points within a specific tolerance.

group_per_stage

Groups the results in a dictionary where the keys are the stage numbers and the values are the coordinates of each point.

group_per_gkey

Groups the results in a dictionary where the keys are the geometric keys and the values are the point coordinates at each stage.

find_points_between_stages

Finds matching points in pairs of neighbouring stages using COMPAS point_in_cloud function.

find_points_from_stage

Finds matching points between one stage and the others using compas point_in_cloud function.

history_to_json

Converts the points history dictionary into json files and saves them in local directory.

Clean Results

find_corrupted_stages

Find the index of the stages with blank points and add them to a list.

find_corrupted_points

Find the keys of the points containing at least one corrupted configuration in their location history.

remove_points_from_results

Remove a list of points form the results.

split_results

Splits the results in chunks.

Evaluate Results

find_abs_max_displacement

Find the absolute maximum displacement value and corresponding key among all the points.

evaluate_displacements

Create a dictionary describing the displacement of a point through successive stages, measured from its initial position, using compas distance_point_point function.
