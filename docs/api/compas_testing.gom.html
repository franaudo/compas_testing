compas_testing.gom

Import/Export

results_to_list

Converts the text results to a list and computes the geometric keys of the points within a specific tollerance.

group_per_stage

Groups the results in a dictionary where the keys are the stage numbers and the values are the coordinates of each point.

group_per_gkey

Groups the results in a dictionary where the keys are the geometric keys and the values are the point coordinates at each stage.

find_points_between_stages

Finds matching points in pairs of neighbouring stages using compas point_in_cloud function.

find_points_from_stage

Finds matching points between one stage and the others using compas point_in_cloud function.

history_to_json

Converts the points history dictionary into json files and saves them in local directory.

Plot in rhino

find_0_stages

Find the index of stages with blank points.

find_0_points

Find the keys of points containing blank coordinates in their location history.

key_to_listcoord

Convert point keys into XYZ coordinates in 3D space

get_test_key

Get a key to test a function.

displ_dic

Create a dictionary describing the displacement of a point through successive stages, measured from its initial position

sliced_dict

Remove unwanted values from a dictionary:

find_max_val

Find the maximal value and corresponding key within a dictionary.

scaled_val_dic

Create a dictionary in which all values are scaled by a given factor

x_rgb

Convert a number between 0 and 1 into an rgb color

draw_stages_colour

Draw point clouds for chosen stages - in Rhino, Point color is defined by its displacement from initial postion

draw_stages

draw the point cloud for input stages

del_key_in_dict

Remove a key and its values from a dictionnary

point_trajectory

Draw the locations in space of a point throughout the successive stages

find_rhpoint_key

Select a point on rhino and get its key in the points_history dictionary
